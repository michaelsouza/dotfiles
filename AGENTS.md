## Agent skills

### Issue tracker

Issues are tracked in GitHub Issues for this repo (`michaelsouza/dotfiles`). External PRs are not treated as a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Default canonical vocabulary: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context repo — `CONTEXT.md` and `docs/adr/` at the root. See `docs/agents/domain.md`.
